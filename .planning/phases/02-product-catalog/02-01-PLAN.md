   - `getAllProducts(limit?: number)` — payload.find products, sort by sortOrder, depth 1
   - `getProductBySlug(slug: string)` — payload.find products where slug equals, limit 1, depth 2, return first doc or null
   - `getProductsByCategory(categorySlug: string, limit?: number)` — first get category by slug, then find products where category equals categoryId, depth 1
   - `getAllCategories()` — payload.find categories, sort by sortOrder, depth 1
   - `getCategoryBySlug(slug: string)` — payload.find categories where slug equals, limit 1, depth 1, return first doc or null
   All functions use `getPayload` from `@/lib/payload`. Each returns typed Payload responses.

2. Create `src/components/ui/Container.tsx` — max-width wrapper:
   ```tsx
   export function Container({ children, className }: { children: React.ReactNode; className?: string }) {
     return <div className={`mx-auto max-w-7xl px-4 sm:px-6 lg:px-8 ${className || ''}`}>{children}</div>
   }
   ```
